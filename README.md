
# FrontEndCRUD

Build a frontend to display, create, edit update delete the posts -


## Tech Stack

**Client:** React

**db:** firebase


## Installation

Install my-project with npm

```bash
  git clone https://github.com/Anoosh-MR/FrontEndCRUD.git
```

```bash
  cd FrontEndCRUD
  cd client
```

```bash
  npm install
```

```bash
  npm run dev
```
